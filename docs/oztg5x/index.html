vue 自定义插件

2022-07-09 06:45

插件是什么

插件通常会给vue添加全局功能。

  • 添加全局方法或属性
  • 添加全局资源，指令，过滤器，过渡等
  • 添加组件选项，通过mixin方法
  • 添加Vue实例方法，添加到Vue.prototype上
  • 一个库，提供自己的API

原理

Vue插件本质是一个对象，有一个公开的**install**属性，值为一个函数，接收两个参数，第一个为**Vue**，第二个为可选的选项对象。最终此对象会被导出供其他人使用。

//所谓vue的插件，就是一个js对象
let myplugin={
    install:function(Vue,Options){
        //在这里写插件需要实现的功能
    }
}
export default myplugin;

示例 vue-cli

  1. 定义插件
import axios from 'axios';
const instance = axios.create({
  baseURL: 'http://localhost:3000',
});
const http = {
  install(Vue) {
    Vue.prototype.$http = instance;
  },
};
export default http;
  1. 使用插件
import http from './http';
import Vue from 'vue';
Vue.use(http);
  1. 在main.js引用
import '@/plugins';
