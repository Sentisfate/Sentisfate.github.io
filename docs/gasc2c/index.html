工程化

2022-07-08 08:06

pnpm 安装配置**npm install pnpm -g**vite的创建项目

  1. 安装vite，**npm install vite -g**
  2. vite 创建项目，**pnpm create vite myapp**
  3. 安装依赖，**pnpm install**
  4. 启动项目，**pnpm run dev**
  5. **改变启动端口 ****--port 端口号**

cnpm 安装配置npm install -g cnpm --registry=https://registry.npm.taobao.org

**npm run serve** 走的是开发环境， 生产环境，**dependencies**，开发环境： **devDependencies**** **

依赖安装**npm i mockjs json-serve -D** 这里是一下装俩**npm i axios -S**生产版本-D 表示装载在**devDependencies** 这个依赖项中，只参与开发环境的打包，不参与生产环境的打包目录中**__mock__**表示是自建文件夹(文件)，不参与打包

配置环境参数

开发版本环境参数配置开发环境接口基础地址**VUE_APP_BASE_API='http://localhost:3000'**

生产版本环境参数配置生产环境接口基础地址**VUE_APP_BASE_API='http://localhost:3000'**

打包 npm run build

生成dist 目录，将项目打包至此处预览打包后项目npm i serve -gserve -s ./dist

通过环境参数封装http请求

script src导入 type=module 可以直接使用import语法export {a} 必须解构导入export default {} 无须解构导入
