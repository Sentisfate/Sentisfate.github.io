WMS服务

2023-06-28 02:01

支持的操作：

  • GetCapabilities 请求服务的元数据
  • GetMap请求地图图像
  • GetFeatureInfo请求关于地图要素的信息
  • GetLegendGraphic请求图例
  • GetStyles请求用户自定义的样式

WMS必须至少支持GetCapabilities GetMap操作，根据服务的元数据来确定支持那些操作.

元数据的类型，不同元数据支持什么操作？

GetCapabilities

可以通过类似下面的地址来获取一个服务的元数据，

http://localhost:8080/geoserver/newSpace/wms?service=WMS&request=GetCapabilities

  • http://localhost:8080/geoserver/Space/wms 请求路径
  • service=WMS 请求的服务类型
  • request=GetCapabilities 请求的操作

  1. service 之间描述的是服务的名称，关键词，等信息
  2. Capability 之间描述的是服务支持的操作及包含的图层
    1. Request 支持的操作
    2. Layer 服务包含的图层，样式信息，图例信息等
    3. DCPType 规定了请求的方式

根据元数据信息，可以使用GetMap进行请求地图

GetMap

http://localhost:8080/geoserver/newSpace/wms?service=WMS&request=GetMap&CRS=EPSG:3857&version=1.3.0&Layers=dem&styles=dem1&bbox=1.2299320322E7,3300526.4354,1.2501170322E7,3520126.4354&width=1200&height=900&format=image/png&transparent=true将上面这串url输入到浏览器中可以得到下面的图片

WMS getMap请求涉及到的字段如表中所示，

  • version The mandatory VERSION parameter is defined in 6.9.1. The value “1.3.0” shall be used for GetMap requests that comply with this International Standard. 最新版应填1.3.0
  • request 为getMap
  • Layers为要请求的图层名称，多个图层,分隔
  • Width 图片宽度，像素
  • Height 高度，像素
  • Styles 样式名称
  • Format 元数据支持的格式
