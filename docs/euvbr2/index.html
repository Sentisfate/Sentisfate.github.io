字符串轮转

2022-09-29 08:51

题目描述

解法1 拼接字符串

思路：

不难发现，s2如果是轮转的字符串的话，s2必定是s1+s1的子串

临界条件： :::info

::: 证明： :::tips

:::

/**
 * @param {string} s1
 * @param {string} s2
 * @return {boolean}
 */
var isFlipedString = function(s1, s2) {
  if(s1.length!=s2.length) return false
  // 字符串s+s包含了所有的s2的轮转的可能，只需要确认是否s2是s+s的子串即可
  return (s1+s1).includes(s2)
};
