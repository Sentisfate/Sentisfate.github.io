2023-04-17 04:20

给点添加标注

点添加标注很简单，直接给实体添加label属性即可。

entity.label = new Cesium.LabelGraphics({
  text: 'abcda',
});

标注被点盖住可以通过设置偏移让它移动到上面。

entity.label = new Cesium.LabelGraphics({
  text: 'abcda',
  pixelOffset: new Cesium.Cartesian2(0.0, -30),
});

给线添加标注

线不能像点一样添加标注，因为点实体自带一个position属性，而线和面实体，position属性是为undefined，标注是依据该属性来确定位置，因此需要计算添加标注的坐标。如何计算，简单点的算法，就是取线的点集的中间那个点为position，然后添加标注。对于点个数为2的倍数的线，计算出中间点即可。

let entity = viewer.entities.add({
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArray([-75, 35, -125, 35]),
    width: 5,
    // arcType: Cesium.ArcType.RHUMB,
    material: Cesium.Color.GREEN,
  },
});
console.log(entity);
let positions = entity.polyline.positions;
console.log(positions._value);
let count = positions._value.length;
console.log(count);
let middle;
if (count % 2 == 1) middle = positions._value[Math.floor(count / 2)];
else {
  let startPos = positions._value[count / 2 - 1];
  let endPos = positions._value[count / 2];
  middle = new Cesium.Cartesian3(
    (startPos.x + endPos.x) / 2,
    (startPos.y + endPos.y) / 2,
    (startPos.z + endPos.z) / 2
  );
}
entity.position = middle;
entity.label = new Cesium.LabelGraphics({
  text: 'abcda',
});

面加标注

面的中心点通过BoundingSphere 这个API来计算

let polygon = viewer.entities.add({
  polygon: {
    hierarchy: Cesium.Cartesian3.fromDegreesArray([
      -115.0, 37.0, -115.0, 32.0, -107.0, 33.0, -102.0, 31.0, -102.0,
      35.0,
    ]),
    material: Cesium.Color.RED,
  },
});

let polygonPositions = polygon.polygon.hierarchy.getValue(
  Cesium.JulianDate.now()
).positions;
let center = Cesium.BoundingSphere.fromPoints(polygonPositions).center;
console.log(center);
polygon.label = new Cesium.LabelGraphics({
  text: 'abc',
  // eyeOffset: Cesium.Cartesian3.ONE,
  // scale: 1.,
});
polygon.position = center;
