三栏布局

2022-09-22 11:39

思路： :::warning 圣杯布局，双飞翼布局，三栏高度不定，中间栏内容多先渲染 ::: :::tips

  • 三栏布局，要求左右两边盒子宽度固定，中间盒子宽度自适应，高度由内容决定
  • 中间盒子先渲染，要放在左右盒子最前面
  • 实现方法：
    1. 定位，子绝父相
    2. 圣杯布局 容器设置margin左右，中间盒子宽度100%，左右中全部浮动，左右盒子设置margin负值，相对定位，左盒子向left负自身宽度，右盒子向left负自身宽度
    3. 双飞翼，容器内只存放中间盒子，左右盒子为容器后面的兄弟节点。中间容器宽度100%，里面的盒子设置左右margin，左右中间(容器)盒子均向左浮动，左盒子左margin设置为-100%，右盒子左margin设置为-自身宽度
    4. flex布局 左边盒子设置order为-1，中间元素设置flex为1 :::
.container {
  margin: 0 120px;
}

.main {
  float: left;
  width: 100%;
  height: 200px;
  background: black;
}

  right: -100px;
  height: 100px;
