语法糖 async和await (ECMAScript7)

async

概念

异步的意思

作用

是一个关键字，用处是将函数变为异步函数，异步函数返回的结果是一个promise对象，里面的结果值是函数的返回值。

// 在同步函数前，添加一个async关键字，就会变成异步函数

async function a2() {
  return 222
}

const b1 = a2()
console.log(b1)//返回的 是一个promise对象，完成状态结果值为222
b1.then(data => console.log(data))

await

概念

等待的意思。

作用

异步函数会暂停执行，等待await修饰的异步操作promise完成后拿到异步操作的结果。

使用

必须要用在异步函数里.async可以单独使用.

(async function (){
    const p=new Promise((reslove,reject)=>{
        reslove(22);
    })
    const b=await p;
})

  1. 编写add.html页面脊梁

  1. 编写edit.html页面

<a href="./edit.html?id=${item.id}">修改</a>
