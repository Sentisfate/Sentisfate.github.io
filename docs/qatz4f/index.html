注意，要让数据变为响应式数据，需要使用变量.value

定义响应式数据流程

//定义响应式数据
let msg = ref('hello')
//修改数据
msg.value='hello world'

经过**ref() **修饰的基础数据是一种**RefImpl**类型的对象，**RefImpl**(reference，implement)**是一种引用的实现。相当于将msg塞到了一个对象里。

如果要再对数据进行操作，就要改变其value属性的值。而复杂数据类型的引用的实现是通过**Proxy** **对象来操作的，能够获得一个对象的代理

value是一个Proxy代理对象。ref函数可以接受基本数据类型和引用类型基本数据类型的响应式是通过 Object.defineProperty()的get和set完成引用数据类型是通过Vue3的reactive函数实现的。

二、ref/reactive/toRefs

ref ->data引用的实现 const list = ref(['html', 'css', 'js']);改变数据要使用 .value 才能实现响应式reactive->data+methods

const state = reactive({
  count: 1,
  add() {
    state.count++;
  },
});
console.log(state);

<div @click="state.add">{{ state.count }}</div>

解构**reactive**，使用**toRefs**

const { count3, add } = toRefs(state);

//或者
const { count3, add } = toRefs(
  reactive({
    count3: 1,
    add() {
      console.log(this);
      this.count3++;
    },
  })
);

使用

import { createStore } from 'vuex';
const store = createStore({
  state() {
    return {
      count: 0,
    };
  },
  mutations: {
    increment(state) {
      state.count++;
    },
  },
});
export default store;

<template>
  <div>
    home
    <button
      @click="
        () => {
          store.commit('increment');
        }
      "
    >
      {{ count }}
    </button>
  </div>
</template>

<script setup>
import { useStore } from 'vuex';
import { computed } from 'vue';

const store = useStore();
console.log(store);
// 结合计算属性
const count = computed(() => store.state.count);
</script>
