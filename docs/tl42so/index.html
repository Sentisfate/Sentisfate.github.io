Ajax，Fetch，Axios的区别

2022-08-16 06:16

互联网历史

  • Web1.0 页面纯显示，不需交互
  • Web2.0 用户可以向页面提交内容,BBS,论坛，Ajax是Web2.0出现的产物Ajax(Asynchronous JavaScript and XML) ，是一种技术的统称
  • Fetch 是一个具体的API，是浏览器的原生的API，和 XMLHttpRequest是一个级别的。
  • Axios 是一个第三方的http请求库

区别

:::info Ajax(概念) - Fetch 具体的API - Axios第三方的http请求库 :::

原生AJax的实现

  1. 创建ajax���核心对象 XMLHttpRequest实例

**var xhr = new XMLHttpRequest()**

  1. 将实例与服务器进行连接

**xhr.open('get', url,)**

  1. 发送请求

**xhr.send()**

  1. 监听响应

**xhr.onreadystatechange = function () {}**

    let url = 'http://122.112.161.135:8000/banner'
    // XMLHttpRequest ajax 的核心对象
    // 创建ajax的核心对象
    function ajax(url) {
      // 创建ajax核心对象
      var xhr = new XMLHttpRequest()
      // 与服务器创建连接
      xhr.open('get', url,)
      // 发送请求
      xhr.send()
      // 响应
      xhr.onreadystatechange = function () {
        console.log(xhr)
        if (xhr.readyState == 4) {
          if (xhr.status == 200) {
            let res = JSON.parse(xhr.responseText)
            console.log(res)
          }
        }
      }
    }
    ajax(url)


let url = 'http://122.112.161.135:8000/banner'
// XMLHttpRequest ajax 的核心对象
// 创建ajax的核心对象
function ajax({ url, type = 'get', data, dataType, success, error }) {
  // 创建ajax核心对象
  var xhr = new XMLHttpRequest()
  // 与服务器创建连接
  xhr.open(type, url)
  // 发送请求
  xhr.send()
  // 响应
  xhr.onreadystatechange = function () {
    if (xhr.readyState == 4) {
      if (dataType == 'json' || dataType == undefined) {
        if (xhr.status == 200) {
          let res = JSON.parse(xhr.responseText)
          success(res)
        }
        else {
          error(res)
        }
      }
    }
  }
}
ajax({
  url, type: 'get', success: (res) => {
    console.log(res)
  }
})

Fetch

浏览器原生API和XMLHttpRequest 是一个级别，相当于XMLHttpRequest的升级版本Fetch语法更简洁，支持Promise

    let url = 'http://122.112.161.135:8000/banner'
    fetch(url).then(async ctx => {
      let res = await ctx.json()
      console.log(res)
    })
    fetch(url).then(ctx => {
      console.log(ctx)
      ctx.json().then(res => {
        console.log(res)
      })
    })

Axios

是一个基于promise的网络请求的库，服务端使用原生的Node.js http，而在客户端使用XMLHttpRequest，客户端axios的特性

  • 从浏览器创建xmlhttprequest
  • 支持promise
  • 拦截请求和响应
  • 取消请求
  • 自动转换json数据
  • 支持XSRF防御
