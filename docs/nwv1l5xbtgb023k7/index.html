getLocation

实现定位功能，具体参数查看官网文档。

const mapCtx=this.mapCtx
wx.getLocation({
  type:'wgs84',
  success(res){
    const lon=res.longitude
    const lat=res.latitude
    mapCtx.moveToLocation({
      longitude:lon,
      latitude:lat
    })
  },
  fail(err){
    console.log(err)
  }
})

注意需要在app.json中配置相关地理位置信息(其他相关地理接口配置见https://developers.weixin.qq.com/community/develop/doc/000a02f2c5026891650e7f40351c01)

{
  "permission": {
    "scope.userLocation": {
      "desc": "你的位置信息将用于小程序位置接口的效果展示" 
    }
  },
  "requiredPrivateInfos": [
    "getLocation"
  ]
}
