类型收窄

2023-06-21 09:31

在处理联合类型时，我们有时会想让该值为其中的一种类型，方便处理。一种常用的做法是进行类型断言，但是治标不治本，因此我们需要进行类型收窄。类型收窄有以下几种方式：

typeof

  • “string”
  • “number”
  • “bigint”
  • “boolean”
  • “symbol”
  • “undefined”
  • “object”
  • “function”
function foo(x:number|string){
  if(typeof x==='number'){
    return Math.sqrt(x)//这里默认会将x认为number
  }
  else x.toUpperCase()//这里则将x认为string
}

真实性收窄

利用if语句进行收窄，排除以下类型

  • 0
  • NaN
  • ""
  • 0n
  • null
  • undefined

}

相等性收窄

switch，===，!==，==，!=可以用于对两个类型进行收窄，即当表达式为真时，左右两个值，或者涉及到的值的类型为满足条件的类型。

function foo(x:number|string,y:number|undefined){
  if(x===y){
    x as number
    y as number
  }
}

in 运算符收窄

in运算符用于确定对象或其原型链上是否具有指定属性。

interface Person{
  name:string
}
interface Dog{
  wang:()=>void
}
function bar(o : Person|Dog){
  if('wang' in o){
    o.wang()
  }
}

instanceof 收窄

instanceof用于检查一个值是否是另一个值的instance，x instanceof Foox的原型链上是否包含Foo.prototype。
