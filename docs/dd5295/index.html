js如何判断数组是否为空

2022-06-10 02:59

提出有如下几种判断方法

  • arr 直接判断 X
  • arr==[] X一直永远为false
  • arr.toString()
  • arr.length<=0
  • arr==false
  • arr[0]
  • arr==’’ 利用了隐式转换

下面写一个demo来检测一下:

// 先来看看空数组是否能转换成bool型
console.log(Boolean([]));
//长度
console.log(Boolean([].length));
// 与一个空数组比较
console.log([] == []);

// 用第一个元素判断
console.log(Boolean([][0]));
// 与字符串
console.log([] == '');
// 转换成字符串
console.log(Boolean([].toString()))
// 与false进行比较
console.log([] == false)
