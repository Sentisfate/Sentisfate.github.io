Stylus CSS预处理器

2022-06-16 06:38

一、介绍

1.是什么

**stylus**是一个css的预先处理框架，简化了CSS语法，是富于表现力、动态的、健壮的css预处理器。

2.语法

  1. 冒号，分号，括号，逗号可选
  2. 变量插值混合书写
  3. 缩进表示层级结构
  4. &连接并集选择，伪类选择

二、如何使用

  1. 全局安装stylus npm i stylus -g
  2. 写一个后缀为**.**styl** **的stylus文件。
//  语法
/* 
    选择器
        属性1：值1
        属性2：值2
        后代选择器
            属性:值
            &并集或者伪类选择器 &表示
                属性：值

 */

*
    margin: 0
    padding: 0

.red
    color: red

li
    list-style: none

a
    text-decoration: none

.top-nav
    height: 40px
    width: 400px
    background: #eee
    .menu-item
        display: inline-block
        padding: 0 10px
        line-height: 40px
        &:hover
            background: #999

        &.current
                a
                    background: #999
                    color: #333
  1. 将stylus文件编译为css文件，**stylus .styl文件路径**

📌这样编译会不会多此一举，不如直接写css文件？ 正常来讲css文件是静态的，stylus可以实现动态的样式效果。此外，语法简洁，写起来速度也很快。实际使用并不会减少效率。

编译成功，会在原来的路径下，多一个同名的css文件

  1. html页面引入css样式。
<link rel="stylesheet" href="./styles/test.css">
  1. 查看页面结果

附上完整代码：

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="./styles/test.styl">
    <!-- 浏览器只认CSS，因此要用stylus把.styl转义为css -->
    <link rel="stylesheet" href="./styles/test.css">
</head>

<body>
    <div class="red">2000</div>
    <nav class="top-nav">
        <ul>
            <li class="menu-item current current">首页</li>
            <li class="menu-item current">产品</li>
            <li class="menu-item current">关于</li>
        </ul>
    </nav>
</body>

</html>
