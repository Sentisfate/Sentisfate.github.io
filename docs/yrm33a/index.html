修剪二叉搜索树

2022-09-10 05:37

题目描述

解法1 dfs

思路：

利用二叉搜索树的性质，左子树小于根节点小于右子树。 对于二叉搜索���而言，修剪后的可以看成为满足条件的根节点，左结点为修剪后的左子树根节点，右节点为修剪后的右子树的根节点。 如果当前结点在范围内，则该结点可作为根节点，左右子树均有可能在范围内。 如果小于low，则左子树均小于low，根节点只可能在右子树中 如果大于hight，则根节点只能在左子树中

/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @param {number} low
 * @param {number} high
 * @return {TreeNode}
 */
var trimBST = function(root, low, high) {
    //对于一个二叉搜索树，左子树小于根节点小于右子树
    //对于一个结点而言有不外乎三种情况
    /**
        1.根节点小于low，此时左子树必定小于low，因此，可能的结点只能在右子树
        2.根节点大于hight，右子树必定大于high，因此，可能的结点只能在左子树。
        3.根节点在范围内，则左右子树都有可能在范围内。因此需要遍历两边，根节点的左结点为修剪后的左子树的根节点，根节点的有结点为修剪后的右子树的根节点，因此需要在递归过程中返回根节点
        在遍历过程中上述三种情况都有可能会出现，因此可以用递归
     */
    const dfs=(root)=>{
        if(root==null){
            return null
        }
        if(root.val<low){
            //返回修剪后的右子树的根节点
            return dfs(root.right)
        }
        if(root.val>high){
            return dfs(root.left)
        }
        //如果当前的结点满足条件，该结点就是当前树的根结点
        //左节点为修剪后的左子树的根结点，右节点为修剪后的右子树的根节点
        root.left=dfs(root.left)
        root.right=dfs(root.right)
        // 返回根节点
        return root
    }
    return dfs(root)

};
