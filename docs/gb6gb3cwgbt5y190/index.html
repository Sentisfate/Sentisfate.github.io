去除cesium默认控件

2022-12-11 02:54

api

new Cesium.Viewer(container, options)

https://cesium.com/learn/cesiumjs/ref-doc/Viewer.html#.ConstructorOptions

默认样式 

...

const config = {
  animation: false, //左下角仪表盘
  baseLayerPicker: false, //基础图层选择控件
  fullscreenButton: false, //全屏控件
  geocoder: false, //地名搜索控件
  homeButton: false, //返回初始位置
  sceneModePicker: false, //二维三维切换控件
  timeline: false, //事件线
  navigationHelpButton: false,
  // selectionIndicator: false, //没看到在哪
  // infoBox: false, //没看到在哪
};

...

const viewer = new Cesium.Viewer('map', config);
//去除版权信息
viewer._cesiumWidget._creditContainer.style.display = 'none';

这样就能得到一个光秃秃的地球了。
