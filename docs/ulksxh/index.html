linux 安装mysql

  1. 安装docker
wget -O /etc/yum.repos.d/docker-ce.repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
yum install docker-ce -y
sudo systemctl start docker //启动docker
docker -v //查看版本号
docker info //查看docker详情
  1. 使用docker安装mysql
    1. 安装mysql **docker pull mysql**
    2. 运行mysql命名容器名称为mysql-service并且设置root账号初始密码为123456
docker run --name mysql_service -e MYSQL_ROOT_PASSWORD=123456 -d -p 10060:3306 mysql --default-authentication-plugin=mysql_native_password

--name 命名容器
-d 后台方式运行
-p 映射端口，10060:3306 暴露的是10060端口，实际mysql运行在3306端口，相当于是将访问10060的程序传递到3306端口
  1. 进入mysql 容器 docker exec -it mysql_service bin/bash
  2. 进入sql mysql -u root -p
  3. 使用Navicat Premium 来远程连接服务器的数据库

配置路由和文件对应。

alias指定的目录是准确的，即location匹配访问的path目录下的文件直接是在alias目录下查找的；
root指定的目录是location匹配访问的path目录的上一级目录,这个path目录一定要是真实存在root指定目录下的；

示例
location /huan/ {
      alias /home/www/huan/;
}
访问http://www.wangshibo.com/huan/a.html实际指定的是/home/www/huan/a.html。
上面的路由可以使用root配置达到同样的效果
location /huan/ {
       root /home/www/;
}

在location /中配置root目录；在location /path中配置alias虚拟目录。
