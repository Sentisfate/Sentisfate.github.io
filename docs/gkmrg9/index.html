js 深入理解

2022-06-15 11:18

一、对象

js语言是面向对象的语言，在js程序员看来，万物皆对象。

:::info 对象是一个客观存在，是用代码对客观事物的一种表达。对象的属性就是客观事物的特征属性。 :::

1.对象赋值

对象作为js的一个数据类型，其可以赋值给其他变量。

const xiaoming={
  name:"xiaoming",
  age:1,
}

对象与普通数据类型不同，是存放到堆中的

像 C 语言这样的底层语言一般都有底层的内存管理接口，比如 malloc()和free()。相反，JavaScript 是在创建变量（对象，字符串等）时自动进行了分配内存，并且在不使用它们时“自动”释放。 释放的过程称为垃圾回收。这个“自动”是混乱的根源，并让 JavaScript（和其他高级语言）开发者错误的感觉他们可以不关心内存管理。

js内存空间分为栈(stack)、堆(heap)、池(一般也会归类为栈中)。 其中栈存放变量，堆存放复杂对象，池存放常量。

基础数据类型，往往保存到栈中，比如 :::info

  • Number

  • String

  • Boolean

  • Undefined

  • Null ::: 引用数据类型，往往保存到堆中，比如 :::info

  • Array

  • Object

  • Function ::: 回到上面的对象赋值问题，对象是object类，值保存到堆中，而变量存放到栈中，因此，变量需要有能获取堆中的值的需求，这一需求通过地址来实现。如下图

2.变量赋值

此时如果我们要将**xiaoming**这个变量赋值变量obj，会发生什么呢。

const obj=xiaoming

没错**xiaoming**中的地址会被赋值给这个变量

这种赋值地址的方式，也叫作引用传值，**浅拷贝，**详情可以看下后面的章节。

3.访问对象属性

:::info 对象的数据其实是属性的数据 ::: 访问对象属性的方法有以下两种：

  • **obj.属性**
  • **obj['属性名']**

注意，当属性名不符合标识符系统时或者是表达式，只能用[]语法

console.log(obj['font-size'])
for(let key in obj){
  console.log(obj[key]) //key是个变量表达式，是当前属性名的字符串形式
}
let key='font'
console.log(obj[key+'-size'])//key 是一个表达式

访问属性的过程，其实就是从栈中的地址向找到对象在堆中的位置再去访问具体的属性。

当对象过于复杂的时候，比如一个对象的属性是一个对象的时候，属性中存放的是另一对象的地址。

const zhangsan={
  name:'zs',
  age:35,
  son:xiaoming,
}

二、浅拷贝

1.什么是浅拷贝

发生了引用传递的就是浅拷贝，前后两个变量本质上是一个对象。

2.特点

  1. 引用赋值
  2. 连锁反应

3.示例

const xiaoming={
  name:"xiaoming",
  age:1,
}
const obj=xiaoming
const zhangsan={
  name:'zs',
  age:35,
  son:xiaoming,
}
obj.name='obj'
console.log(xiaoming.name)
console.log(xiaoming===obj)//true

三、深拷贝

1.什么是深拷贝

实际上就是新建了一个对象，不过这个对象和原来的对象全部一模一样。

2.特点

  1. 新的对象
  2. 相互独立

3.示例

const obj = {
    name: 'xiaom',
    age: 12,
    son: {
        name: 'xxm',
    },
    arr: [1, 2, 3, 4, 5],
    unkonw: null,
}
// 深拷贝
function deepCopy(obj) {
    // 排除其他数据类型和null，null也是object
    if (typeof obj != 'object' || obj == null) {
        return obj
    }
    // 定义一个空对象，用于添加属性
    const objCopy = {}
    for (let key in obj) {
        // 严格对象检查
        console.log(Object.prototype.toString.call(obj[key]))
        if (Object.prototype.toString.call(obj[key]) == '[object Array]') {
            objCopy[key] = []
            //遍历数组
            for (let value of obj[key]) {
                // 深拷贝数组的元素
                objCopy[key].push(deepCopy(value))
            }
            continue
        }
        //数组会在上一个语句中进入下次循环，这里可以保证是非数组对象
        if (typeof obj[key] == 'object') {
            objCopy[key] = deepCopy(obj[key])
            continue
        }
        objCopy[key] = obj[key]
    }
    return objCopy
}
xiaoming = deepCopy(obj)
console.log(xiaoming)

四、函数的传参

本质就是变量的赋值过程

1.值传递

太简单了，形参实参互不影响。

2.引用传递

参考前面对象的赋值过程，本质是传递了地址。

3.函数名本身是一个引用

function foo(a) {
  console.log(a)
  a.name = 'xiaoming'
}

// 函数名是一个引用
const b = {
  name: 'xiaomei'
}
foo(b)
console.log(b.name)

function foo2(a) {
  a('123')
}
let b2 = function (c) {
  console.log(c)
}
foo2(b2)

五、函数的返回值

调用函数本质上是一个函数表达式，只要是表达式就有返回值

1.无return情况

返回undefined。

console.log((() => { })())

2.有return情况

返回return语句的结果。

function bar() {
  return {
  }
}
//  {} != {}
console.log(bar() === bar())

3.闭包

当A函数返回了一个B函数的情况下，且B函数还用到了A函数的数据，就会发生产生闭包，此时A函数为闭包。 什么是闭包，简单来讲就是，一个定义在函数内部的函数，且访问了外部函数的变量。

function final(a) {
  return function () {
    //const a = 222 //加上这一句就不会形成闭包
    console.log(a)
  }
}
final(1)()
