GL_INVALID_OPERATION: Uniform type does not match uniform method.

uniform变量的类型与启用类型的方法不一致。WebGLRenderingContext.uniform[1234][fi][v]()WebGLRenderingContext.uniformMatrix[234]fv()

INVALID_OPERATION: uniformMatrix3fv: location is not from current program

类似下面的错误，

第一个是由于在获取全局变量之前，没有调用gl.useProgram(program)，当前gl没有指定program导致的，attribute好像不受影响。第二个问题是同样没有调用gl.useProgram(program)导致，或是类似的充值了空的program等，此时program没有着色程序，自然会报这个错误。解决方案：

  • 对于只有一个program，则在成功创建该程序时就调用gl.useProgram(program)
  • 对于多个program，需要明确每个program对应的着色器程序以及变量的用途，在使用对于的着色器程序时调用。
function createProgram(gl, vsShader, fsShader) {
  const program = gl.createProgram();
  gl.attachShader(program, vsShader);
  gl.attachShader(program, fsShader);
  gl.linkProgram(program);
  if (gl.getProgramParameter(program, gl.LINK_STATUS)) {
    gl.useProgram(program);
    return program;
  }
}
