文件下载

2023-02-13 09:57

思路

利用a标签下载文件

  1. js创建一个隐藏的a标签
  2. 获得资源的url
  3. 设置a标签的**href=url**和**download=filename**属性
  4. 触发**a.click()**
  5. 删除a标签
function download(url, filename) {
  const a = document.createElement('a')
  a.download = filename
  a.href = url
  a.click()
}

跨域下载

当发生跨域时，download设置无效，浏览器会直接打开图片。

此时的解决思路：发起请求获得blob对象，将blob转为url，再通过a标签下载

blob转url

function download(url, filename) {
  fetch(url).then(async ctx => {
    console.log(ctx)
    let res = await ctx.blob()
    console.log(res)
    var newUrl = URL.createObjectURL(res);
    var a = document.createElement('a');
    a.setAttribute('href', newUrl);
    a.setAttribute('target', '_blank');
    a.setAttribute('download', filename); // 自定义文件名（有效）
    document.body.appendChild(a);
    a.click();
    document.body.removeChild(a);
  	URL.revokeObjectURL(url)
  })
}

function download (url, fileName) {//跨域文件路径、下载到本地的文件名
  const x = new XMLHttpRequest();
  x.open("GET", url, true);
  x.responseType = 'blob';
  x.onload=function(e) {
    const url = window.URL.createObjectURL(x.response)
    const a = document.createElement('a');
    a.href = url
    a.download = fileName;
    a.click()
    a.remove
    URL.revokeObjectURL(url)
  }
  x.send();
},

// message 为antd 的一个全局消息提示

这个方法确实可以实现下载。但是也存在一些问题。如果文件过大，可能会导致存在一定延迟(获取整个blob也需要时间)，导致用户体验变差。
