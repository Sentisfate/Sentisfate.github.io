yaxis

2023-03-17 00:56

y轴组件，一般最多有两个，多余两个要配置offset

id string

组件id

show boolean

是否显示

gridIndex number

alignTicks boolean

自动对齐刻度

position 'left'|'right'

offset number

y轴偏移，注：若未将 yAxis.axisLine.onZero 设为 false , 则该项无法生效

type 'value'|'category'|'time'|'log'

坐标轴类型category一般会自动从dataset读取，也可以主动设置yaxis.datatime 时间轴，适用于连续的时序数据。log 对数轴

name string

坐标轴名。

nameLocation 'start'|'middle'|'end'

nameTextStyle object

color

颜色

fontStyle 'normal'|'italic'|'oblique'

字体风格

fontWeight 'normal'|'bold'|'bolder'|'lighter'|number

字体粗细

fontSize 'number'

大小

align 'left'|'center'|'right'

水平对齐

verticalAlign 'top'|'middle'|'bottom'

水平对齐

lineHeight number

行高

backgroundColor string|object

string 为颜色object为背景图片如：

backgroundColor: {
    image: 'xxx/xxx.png'
    // 这里可以是图片的 URL，
    // 或者图片的 dataURI，
    // 或者 HTMLImageElement 对象，
    // 或者 HTMLCanvasElement 对象。
}

borderColor color

borderWidth number

borderType 'solid'|'dashed'|'dotted'

borderDashOffset number

borderRadius number|Array

padding number|Array

shadowColor
