说一说Promise是什么

2022-09-17 13:35

思路 :::tips 微任务，异步，三种状态，then，catch，Promise.resolve()，Promise.reject().Promise.all() Promise.any() Promise.race() ::: :::info 是异步微任务，解决了异步多层嵌套回调（回调地狱）的问题。

  • 是ES6新增的构造函数，可以创建一个promise对象
  • 这个构造函数接收一个函数参数，这个函数有两个参数，分别为resolve，reject，均为函数，前者调用，该对象状态变为fulfilled，后者调用状态变为rejected，均为调用状态为pendding，除pendding外，���态改变后就不会再发生改变。resolve和reject的参数可以传递结果值
  • then可以分别指定成功的回调和失败的回调，并获得结果值。
  • catch可以捕获错误
  • then和catch的返回值均为promise对象可以链式调用
  • Promise的其他方法
    • .resolve，.reject() 返回的promise对象状态为fulfilled,reject，并且获得结果值，由传参决定
    • Promise.all([p1,p2,…])，接收一个可以迭代的promise对象集合，当p1…都成功时，才返回一个成功的promise对象，有一个失败的话就，返回失败对象
    • any([p1,])，有一个成功就返回成功，否则失败
    • Promise.race(iterable)，返回的对象的状态和结果值由参数中最先成功或失败的promise对象决定。且该对象一旦状态改变，立马返回相应的promise对象 :::
