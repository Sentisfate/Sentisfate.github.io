element-ui 表单校验自定义校验，callback() 函数必须保证校验正确的时候有执行，否则，无法校验

// 自定义校验规则
var validateCato = (rule, value, callback) => {
    if (value === '') {
        callback(new Error('请输入分类名'));
    } else if (value == this.form.category) {
        callback(new Error('请输入不一样的分类名'));
    } else {
        // 查询是否有同名的分类
        this.$http
            .get('/categories/name', {
            params: { category: this.form.newCategory },
        })
            .then((res) => {
            const { result } = res;
            if (result[0]) {
                callback(new Error('分类名已存在'));
                return;
            }
            callback();
        });
    }
};

rules: {
    newCategory: {
        validator: validateCato,
            trigger: 'blur',
    },
},

如何不通过脚手架在html页面来使用element-ui，步骤如下：

  1. 在线引入element-ui的样式库。
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  1. 引入vue.js，此步必须在步骤3之前。
<script src="https://cdn.staticfile.org/vue/2.4.2/vue.min.js"></script>
  1. 引入element-ui 组件库，组件这一概念是在vue中实现的，因此要在引入vue后才引入。
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
  1. 示例
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <!-- 引入样式 -->
    <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
    <script src="https://cdn.staticfile.org/vue/2.4.2/vue.min.js"></script>

    <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>

<body>
    <div id="app">
        <el-progress type="circle" :percentage="num"></el-progress>

        </el-upload>
    </div>

    <script>

                return {
                    visible: false,

                this.fun()
            },
            methods: {
                fun() {
                    if (this.num >= 100) return
                    setTimeout(() => {
                        this.num += 20
                        this.fun()
                    }, 1000)
                }
            }
        })
    </script>
</body>

表单 label里输入空格的方法

直接输入空格只能保留一个空格，输入&nbsp;也无法识别，正确的写法：

<el-form-item label="" prop="password">
  <label slot="label">密&nbsp;&nbsp;&nbsp;&nbsp;码</label>
  <el-input
            v-model="form.password"
            type="password"
            placeholder="请输入密码"
            ></el-input>
</el-form-item>
