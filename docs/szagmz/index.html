iconfont使用教程

2022-07-28 14:04

  1. 选择图标加入项目，并下载到本地

  1. 解压后有以下这些文件，需要的只有红框框起来的文件

  1. iconfont.css文件详解
  • **@font-face**定义了一个字体族iconfont，以及规定了实现图标需要引用的文件，分别是上图中的**iconfont.tff,**``**iconfont.woff,**``**iconfont.woff2**，这三个文件要和iconfont.css处于同一目录下（但实测，只需有tiff一个文件也可以实现图标，虽然控制台会报错找不到前面两个文件😵……）。
@font-face {
  font-family: 'iconfont'; /* Project id 3554232 */
  src: url('iconfont.ttf?t=1659015961828') format('truetype');
}
  • .iconfont 是图标实现必须引用的样式，可以在此处修改图标默认的样式，如大小颜色等
  • 后面的则是图标对应的样式，用哪个图标就引用哪个样式。

  1. 实际使用，引入整个iconfont.css文件，然后使用对应的图标样式
...
<head>
...
  <link rel="stylesheet" href="./vue3-pinia/src/assets/styles/iconfont/iconfont.css">
</head>
<body>
  <i class="iconfont icon-shouye"></i>
</body>
...
