获取 Primitive(s)的包围球

2023-07-05 14:10

function getBSFromPrimitive(primitive:Primitive){
  if(!primitive.geometryInstances){
    return BoundingSphere.fromBoundingSpheres(primitive._boundingSpheres)
  }
  let geometryInstances:GeometryInstance[]=[]
  if(!Array.isArray(primitive.geometryInstances)){
    geometryInstances.push(primitive.geometryInstances)
  }else{
    geometryInstances=primitive.geometryInstances
  }
  const boundingSpheres=geometryInstances.map((item)=>{
    return item.geometry.boundingSphere
  })
  return BoundingSphere.fromBoundingSpheres(boundingSpheres)
}
function getBSFromPrimitives(primitives:PrimitiveCollection):BoundingSphere{
  const bsps=[]
  for(let i=0;i<primitives.length;i++){
    const p=primitives.get(i)
    if(p instanceof PrimitiveCollection){
      bsps.push(getBSFromPrimitives(p))
    }else{
      bsps.push(getBSFromPrimitive(p))
    }
  }
  return BoundingSphere.fromBoundingSpheres(bsps)
}
viewer.camera.flyToBoundingSphere(getBSFromPrimitives(viewer.scene.primitives))

用primitive绘制图形的时候，有这样一段代码：

new Geometry({
  attributes: geometryAttributes,
  indices: new Uint32Array(indices),
  primitiveType: primitiveType,
  boundingSphere: BoundingSphere.fromVertices(positions),
});

这里可以看到包围球是在Geometry上的，按照primitive的组成关系，获取包尾球可以通过下面的路线。

需要注意的是，为了节约内存，primitive的releaseGeometryInstances属性默认为true，即geometryInstances会被释放，从primitive上访问不到���属性，为undefined，而能访问_boundingSpheres,每个geometryInstance的包围球。如果能访问到geometryInstances属性时（releaseGeometryInstances为false时，或者在释放geometryInstances之前，其他地方引用了该属性如创建primitive后就立即同步操作使用了该属性），_boundingSpheres 数组长度为0.
