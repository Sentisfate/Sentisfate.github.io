更改geojson(entity)的属性

2023-03-24 06:37

在加载GeoJson的时候，可以配置一个全局的样式，如点，线，面的颜色，填充色，线宽，点的大小，面的边框等

但有些需求让对这些属性进行更改，或者修改其中某一个点的颜色等，那么如何优雅的对它进行修改呢，本文就来提供一个大致思路和实现。

geojsonDataSource的构成

geojsonDataSource的每一个矢量数据是一个entity，在其entities.values字段里。

因此可以通过遍历该数组的方法来获取，修改每一个entity的属性。问题也就转换成修改实体的属性。

//就加载了一个geojson，默认索引为0
let entities = viewer.dataSources.get(0).entities.values;
for (let i = 0; i < entities.length; i++) {
  entities[i].polygon.material = Cesium.Color.fromRandom({
    alpha: 1,
  });
}

entity属性的修改

一个实体的类型可以是很多种的一种，如：点point、平面Plane、墙Wall等，要确定一个entity的类型其实还是挺麻烦的，因为要依次去判断。

而一个geojson的type字段必定为下面之一：“Point”, “MultiPoint”, “LineString”, “MultiLineString”, “Polygon”, “MultiPolygon”, “GeometryCollection”, “Feature”, 或者 “FeatureCollection”。这些类型转换到entity中，无外乎就是点线面而已。因此对于geojson生成的entity，只需判断polygon，point**billboard**，polyline。接下来目标就非常明确了，修改polygon，point，polyline的属性了。

样式修改

在这之前，要先解决上面的一个问题，就是Cesium加载的点数据，竟然都是billboard，这合理吗，显然不合理，因此要把它全部改成point才对。

dataSource.load(geojson).then((res) => {
  console.log(res);
  for (let i = 0; i < res.entities.values.length; i++) {
    let entity = res.entities.values[i];
    if (entity.billboard) {
      entity.billboard = undefined;
      entity.point = new Cesium.PointGraphics({
        color: Cesium.Color.ALICEBLUE,
        pixelSize: 10,
      });
    }
  }
});

Point

点的样式无非这几种，点的图标，点的大小，点的颜色甚至点的坐标。对应字段分别为：

  • color 点的颜色
  • outlineColor 轮廓颜色
  • outlineWidth 轮廓宽度
  • pixelSize 点的大小
entity.point = new Cesium.PointGraphics({
  color: Cesium.Color.ALICEBLUE,
  pixelSize: 10,
  // outlineColor: Cesium.Color.BLUE,
  // outlineWidth: 10,
});
