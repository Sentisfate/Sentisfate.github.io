790 多米诺和托米诺平铺

2022-11-12 03:41

思路： :::info 对于从左向右铺前i列，第i列前的瓷砖全部铺满，则第i列有如下四种状态：

  1. 全空
  2. 上空
  3. 下空
  4. 全满

使用dp[i][j]来记录第i列状态为**j+1**的所有铺法。则

  • 对于状态1，想要变为这种状况，第i-1列必然为全满，即**dp[i][0]=dp[i-1][3]**
  • 对于状态2，第i-1列可以由两种情况组成，第i-1列全空时，铺一个**L**型或者第i-1列下空时，铺一个**_**则**dp[i][1]=dp[i-1][0]+dp[i-1][2]**
  • 对于状态3，同2，**dp[i][1]=dp[i-1][0]+dp[i-1][1]**
  • 对于状态4，前一列上空或者下空，或者前一列全满，竖放一块，或者前一列全空，横放两块。**dp[i][3]=sum(dp[i-1])**

初始情况，需要对第一列进行赋初值。**dp[1][0]=1,dp[1][1]=0,dp[1][2]=0,dp[1][3]=1** :::

/**
 * @param {number} n
 * @return {number}
 */
var numTilings = function(n) {
    let mod=10**9+7
    let last=[1,0,0,1]
    let target=new Array(4).fill(0)
    for(let i=1;i<n;i++){
        target[0]=last[3]%mod
        target[1]=(last[0]+last[2])%mod
        target[2]=(last[0]+last[1])%mod
        target[3]=(last[0]+last[1]+last[2]+last[3])%mod
        last=null
        last=target
        console.log(last)
        target=new Array(4).fill(0)
    }
    return last[3]
};

法二 找规律

:::info 不难发现**res[n] = 2 * res[n - 1] + res[n - 3]** :::

/**
 * @param {number} n
 * @return {number}
 */
var numTilings = function(n) {
    let mod=10**9+7
    let dp=[1,2,5,0]
    if(n<=3) return dp[n-1]
    for(let i=3;i<n;i++){
        dp[3]=(dp[2]*2+dp[0])%mod
        dp[0]=dp[1]
        dp[1]=dp[2]
        dp[2]=dp[3]
    }
    return dp[3]%mod
};
