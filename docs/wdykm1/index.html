灯泡开关

解法1

思路：

对于一个灯泡而言，只存在两种状态开1关0，当对开着的灯泡进行开关时，操作次数为奇数时，灯泡为0，操作次数为偶数时为1相当于没按。此外，题中对灯泡的操作方式共有四类，分别为

  1. 对所有的操作，周期为1
  2. 对偶数操作，周期为2
  3. 对奇数操作，周期为2
  4. 对从1开始，公差为3的数列进行操作，周期为3

因此，不管怎样，最终应该是一个以6为周期的状态结果。以前6个为例进行研究

观察开关1，2，3，4各对哪些灯泡进行了影响，

  • 第一个灯泡受到了1，3，4的影响
  • 第二个灯泡收到了1，2的影响
  • 第三个受到1，3的影响
  • 第四个受到了1，2，4的影响
  • 第五个和第三个状态一致，第6个和第2个状态一致

因此，可以只考虑前4个灯泡即可。这里最多就可以16种情况了。假设按压各开关的次数分别为a,b,c,d

观察到，当d为偶数时，灯泡4与灯泡2的状态相同，灯泡1的状态与3相同，因此，可以通过前三个灯泡来推断第四个灯泡。即1，3相同时，4与2相同，1，3不同时，4与2相反。结果状态最多只有8种。

初始状态为111

  • 按1次时，最多有4种
  • 按2次时，有7种，少了011
  • 按3次即以上时，有8种

临界条件： :::info

  • 当n为1时，为两种，只有开或关
  • 当n为2时，按压1次可以只能获得3种结果，00，10，01，为2次以上时，有4种
  • 当n为3时，根据按压的次数
  • press为0时，返回1 ::: 证明： :::tips

/**
 * @param {number} n
 * @param {number} presses
 * @return {number}
 */
var flipLights = function(n, presses) {
    if(presses==0){
        return 1
    }
    if(n==1){
        return 2
    }
    if(n==2){
        if(presses==1){
            return 3
        }
        return 4
    }
    if(presses==1){
        return 4
    }
    if(presses==2){
        return 7
    }
    return 8
};
