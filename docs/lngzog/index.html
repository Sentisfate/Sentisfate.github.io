纯函数

2022-08-20 11:22

概念

函数返回结果只依赖于它的参数，且在执行过程里没有副作用，就将这个函数叫做纯函数

理解

只依赖参数

let b = 0;
function foo(a) {
  return a + b;
}

function foo(a) {
  return a ;
}

也就是说，纯函数的结果返回值是可预料的，不受外界影响的。闭包就肯定不是纯函数。

没有副作用产生

什么是副作用

副作用就是会产生外部程序可观察到的变化

const position = {
  x: 10,
  y: 5,
};
function bar(position) {
  position.x = position.x * 2;
  return position.x + position.y;
}

好处

使用纯函数不会产生预料之外的情况发生，函数的耦合度低，测试调试非常方便

总结

一个函数的返回结果只依赖于它的参数，并且在执行过程里面没有副作用，我们就把这个函数叫做纯函数。
