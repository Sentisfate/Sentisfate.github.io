相似度为k的字符串

2022-09-21 04:49

题目描述

解法1 BFS

思路：

不妨对s1进行交换操作，我们对s1的每个字符c1进行比较操作，如果和s2的字符c2相等的话，就跳过，比较下一个字符，如果不相等，就从当前位置向后找一个与之相等的字符，记这个字符的位置为pos。此时可能会有多个与c2相等的字符，显然不能对每个都进行选择，因此需要进行限定

  • 不能交换已经匹配好的字符

我们用一个队列来存储每一个选择，即交换后的字符串s1，并记录下s1下一次开始遍历的位置，因为前面的字符已经匹配好了，不用再重复遍历，对队列中的s1进行不断重复上述的选择，直至交换后全部匹配。此时大概就形成了一个树的结构。

也就是说问题可以转换成求从上到下最短的一条路径。所求结果就是叶子节点的最小的深度。 对上述问题进行广度优先搜索，在搜索过程中，若遇到某个节点的字符串已经出现过一次了，可以进行跳过��因此还可以用一个集合来存储已经访问过的节点。

临界条件： :::info

::: 证明： :::tips

:::

/**
 * @param {string} s1
 * @param {string} s2
 * @return {number}
 */
var kSimilarity = function (s1, s2) {
  const swap = (s, leftIndex, rightIndex) => {
    return (
      s.slice(0, leftIndex) +
      s[rightIndex] +
      s.slice(leftIndex + 1, rightIndex) +
      s[leftIndex] +
      s.slice(rightIndex + 1)
    );
  };
  const visit = new Set();
  visit.add(s1);
  // 节点队列，对访问的节点依次入队，并从队首开始进行遍历节点操作
  //这里初始化为字符串s1和开始位置0
  const queue = [[s1, 0]];
  const n = s1.length;
  let k = 0;
  if (s1 === s2) return k;
  while (queue.length) {
    // 每层的长度
    const len = queue.length;
    for (let i = 0; i < len; i++) {
      let [s, pos] = queue.shift();
      while (s[pos] == s2[pos] && pos < n) {
        pos++;
      }
      // 此时s[pos]和s2[pos]是不相等的
      //往后查找可以交换的字符
      for (let i = pos + 1; i < n; i++) {
        if (s[i] == s2[i]) continue;
        // 找到交换的元素后，将交换后的s1入队
        if (s[i] == s2[pos]) {
          const next = swap(s, pos, i);
          if (next === s2) return k + 1;
          if (!visit.has(next)) {
            visit.add(next);
            queue.push([next, pos + 1]);
          }
        }
      }
    }
    k++;
  }
};
