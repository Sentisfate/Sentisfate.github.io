进行二维空间变换前，最重要的就是明确你所处的坐标系以及变换在哪个坐标系进行。不同的坐标系有不同的效果

平移

对于平移，利用glsl的向量的特性，可以简单的使用一个vec2向量来表示各分量的变化。假定位移量以屏幕坐标为准，着色器更改如下：

attribute vec2 a_position;
uniform mat3 u_matrix;
//位移量
uniform vec2 u_translation;
varying vec4 v_color;
void main() {
  vec2 position=a_position+u_translation;// 在屏幕坐标系下进行平移
  gl_Position = vec4((u_matrix * vec3(position, 1)).xy, 0, 1);
  v_color = gl_Position * 0.5 + 0.5;
}

旋转是图形中的每个点，绕某个确定的点进行旋转，利用三角函数可以计算最新的点，对于绕原点旋转的图形，公式表现如下：

注意上面的矩阵角度相对于坐标系逆时针为正，如果想要以顺时针为正计算，用

代入即可。对于正常的笛卡尔坐标系，逆时针顺序是按照1、2、3、4象限进行的，与人眼观察一致。而对于屏幕坐标系这类y轴翻转了的直角坐标系，其1、2、3、4象限顺序在人眼观察看来是顺时针。因此，在进行图形旋转前，要先确定当前点的坐标是在什么样的坐标系下进行的，如下面这几种情况：

  1. 对像素坐标进行旋转

对像素坐标用旋转矩阵，则其会绕屏幕左上角旋转，在着色器中代码大致如下：

attribute vec2 a_position;
// 像素坐标转裁剪空间转换矩阵
uniform mat3 u_matrix;
// 旋转
uniform mat2 u_rotate;
// canvas的分辨率
uniform vec2 u_resolution;
void main() {
  // 旋转
  vec2 position=u_rotate*a_position;
  // 将坐标转换为裁剪空间坐标
  position=(u_matrix * vec3(position, 1)).xy;
  gl_Position = vec4(position, 0, 1);
}
  1. 对裁剪空间坐标进行旋转，则是在将点的坐标转换为裁剪空间后进行旋转，旋转中心为裁剪空间坐标原点
...
vec2 position= (u_matrix* vec3(a_position,1)).xy;
// 后旋转
position=u_rotate*position;
...
  1. 或者指定某一个点进行旋转，这里以屏幕坐标下的图形中心点为例。
attribute vec2 a_position;
// 像素坐标转裁剪空间转换矩阵
uniform mat3 u_matrix;
// 位移量
uniform vec2 u_translation;
// 旋转
uniform mat2 u_rotate;
// 缩放
uniform mat2 u_scale;
// canvas的分辨率
uniform vec2 u_resolution;
uniform vec2 u_rotateCenter;
varying vec4 v_color;
void main() {
  //先将坐标转换为以u_rotateCenter 为原点的坐标
  vec2 position=a_position-u_rotateCenter;
  // 对变换后的坐标进行旋转
  position=u_rotate*position;
  // 将坐标转为屏幕坐标
  position=position+u_rotateCenter;
  // 将旋转后的坐标转换为裁剪空间坐标
  position=(u_matrix * vec3(position, 1)).xy;
  gl_Position = vec4(position, 0, 1);
  v_color = gl_Position * 0.5 + 0.5;
}

或者先变换到裁剪空间再旋转，注意和前面的旋转方向（肉眼观察，如果不改传入的旋转矩阵的话）就不一样了

vec2 rotateCenter=(u_matrix * vec3(u_rotateCenter, 1)).xy;
vec2 position=(u_matrix * vec3(a_position, 1)).xy;
// 移动坐标系旋转，再恢复坐标系
position=u_rotate*(position-rotateCenter)+rotateCenter;

缩放

同样用矩阵表示，如下：

与旋转一样，不同的参考中心对缩放的效果肯定也不一样。这里以屏幕坐标原点和指定参考中心为例，看下效果。指定参考中心：

attribute vec2 a_position;
uniform mat3 u_matrix;
// 位移量
uniform vec2 u_translation;
// 旋转
uniform mat2 u_rotate;
// 缩放
uniform mat2 u_scale;
// canvas的分辨率
uniform vec2 u_resolution;
// 图形中心-像素坐标系
uniform vec2 u_center;
varying vec4 v_color;
void main() {
  // 缩放
  vec2 position=u_scale*(a_position-u_center)+u_center;
  position=(u_matrix * vec3(position, 1)).xy;
  gl_Position = vec4(position, 0, 1);
  v_color = gl_Position * 0.5 + 0.5;
}
