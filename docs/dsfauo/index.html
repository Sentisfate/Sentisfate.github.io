最长数对链

2022-09-03 03:32

题目描述

思路，要想要数对链最长，要保证数对的第二个元素小于下个数对的第一个元素。

解法一、贪心

对于一个数对链，始终发现该链中的数对的第二个元素是按升序排列的，第一个元素均大于前一个数对的第二个元素。因此希望先通过第二个元素进行升序排列，得到的数组就直接是最大数对链。遍历该数对链，将不符合条件的数对舍去即可

/**
 * @param {number[][]} pairs
 * @return {number}
 */
var findLongestChain = function (pairs) {
  // 将数组排序
  pairs.sort((a,b)=>a[1]-b[1])
  //指向带判断的数对的第二个元素
  let cur=pairs[0][1]
  let count=1
  //逐个判断每个数对的第二个元素和下一个数对的第一个元素是否符合条件
  for(let i=1;i<pairs.length;i++){
      //符合条件，就计数加一，并且指针移到符合条件的数对的第二个元素
      if(cur<pairs[i][0]){
          count++
          cur=pairs[i][1]
      }
  }
  return count
};

解法二、动态规划

思路，每个数对都可能是符合条件的数对链的结尾，从中找出最长的一个即可。此外以每个数对结尾的数对链也有若干中可能，需要找处这些可能中最长的一个。

  • 使用一个数组记录以数对i结尾的链的最大长度
  • 将数对数组进行排序
  • 遍历数对
  • 符合条件的数对链中的数对只可能在当前数对**i**的前面
  • 遍历前面的数对，找出符合条件的数对**j**，则数对**i**结尾的链的长度就可能为以数对**j**结尾的链的长度**+1**找出其中的最大值
  • 找出记录长度的数组最大值
var findLongestChain = function (pairs) {
    // 先将pairs进行排序
    pairs.sort((a,b)=>a[1]-b[1])
    console.log(pairs)
    // 记录以pairs[i]为末尾的链的长度,最短为1
    let arr=new Array(pairs.length).fill(1)
    // 遍历每个数对，链中第i个数对前面的数对只能在i前面
    for(let i=0;i<pairs.length;i++){
        for(let j=0;j<i;j++){
            // 当第j个数对符合以数对i结尾时，以数对j结尾的链都符合条件，数对i的链的长度可以是数对j的长度加1
            if(pairs[j][1]<pairs[i][0]){
                //从中取出符合条件的数对的最大值即可
                arr[i]=Math.max(arr[j]+1,arr[i])
            }
        }
    }
    return Math.max(...arr)
};
