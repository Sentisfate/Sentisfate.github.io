获取地图服务遇到的问题

2023-06-28 05:58

获取元数据

获取元数据方面可以通过http://localhost:8080/geoserver/wms?service=WMS&request=GetCapabilities查询所有未隔离的工作空间下的元数据，或者通过http://localhost:8080/geoserver/工作空间名/wms?service=WMS&request=GetCapabilities查询指定工作空间下的元数据。元数据指已经发布的图层：

未隔离的工作空间指Isolated Workspace未勾选

GetMap

元数据有两种方式，则地图获取同样可以有两种方式，区别在于http://localhost:8080/geoserver/工作空间名/wms?service=wms&...&Layers=图层名http://localhost:8080/geoserver/wms?service=wms&...&Layers=工作空间名:图层名这些其实已经在获取到的元数据中有所体现了。

此外，注意在请求的时候，bbox与CRS对应范围应该一致，不能用经纬度范围表示一个投影系下的范围，因为实际坐标范围不一致。
