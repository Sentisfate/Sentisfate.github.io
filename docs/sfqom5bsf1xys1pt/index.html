绑定与赋值

对于对象和数组，有两种解构模式，绑定模式，赋值模式。

绑定

需要属性和变量名对应或者重命名。可以嵌套解构

const obj = { a: 1, b: { c: 2 } };
const { a, b: { c: d } } = obj;

此时a，d都为只读变量，如果需要调整为可写，需要重新let声明

const { a } = obj; // a is constant
let { b: { c: d } } = obj; // d is re-assignable

赋值

const numbers = [];
const obj = { a: 1, b: 2 };
({ a: numbers[0], b: numbers[1] } = obj);

默认值

当属性为undefined或不存在时，会取到默认值

const [a = 1] = []; // a is 1
const { b = 2 } = { b: undefined }; // b is 2
const { c = 2 } = { c: null }; // c is null

剩余属性

剩余属性（**...rest**）结束解构模式。此模式会将对象或数组的所有剩余属性存储到新的对象或数组中。

const { a, ...others } = { a: 1, b: 2, c: 3 };
console.log(others); // { b: 2, c: 3 }

const [first, ...others2] = [1, 2, 3];
console.log(others2); // [2, 3]

妙用

交换变量

let a = 1;
let b = 3;

[a, b] = [b, a];

const arr = [1, 2, 3];
[arr[2], arr[1]] = [arr[1], arr[2]];
console.log(arr); // [1, 3, 2]

解构返回值，忽略返回值

function f() {
  return [1, 2, 3];
}

const [a, , b] = f();
console.log(a); // 1
console.log(b); // 3

const [c] = f();
console.log(c); // 1

解构可迭代对象

const [a, b] = new Map([[1, 2], [3, 4]]);
console.log(a, b); // [1, 2] [3, 4]

// 不可迭代对象不能解构
const obj = { 0: "a", 1: "b", length: 2 };
const [a, b] = obj;
// TypeError: obj is not iterable
