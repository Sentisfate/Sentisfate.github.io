    if (cur == 1) {

2.Array.from

对一个类似数组或可迭代对象创建一个新的，浅拷贝的数组实例。语法: **Array.from(**_**arrayLike**_**[, **_**mapFn**_**[, **_**thisArg**_**]])**参数：

  • **arrayLike **想要转换成数组的伪数组对象
  • **mapFn** 一个回调函数，新数组的每个元素都会传入作为形参来执行该函数
  • **thisArg **执行**mapFn**的this指向
