vue 集成 echarts入门

一、集成echarts

  1. 安装

**npm install echarts --save**

  1. js引入

_**import**_** * **_**as**_** echarts **_**from**_** 'echarts/core';**上面的代码会引入所有的图表和组件按需引入**import {BarChart} from 'echarts/charts';**

  1. html引入

**<script src="echarts.min.js"></script>**](https://cdn.bootcss.com/echarts/4.2.1-rc1/echarts.min.js">)二、常规使用

  1. 快速上手
    1. 引入echarts

**<script src="**[**https://cdn.bootcss.com/echarts/4.2.1-rc1/echarts.min.js"></script>**](https://cdn.bootcss.com/echarts/4.2.1-rc1/echarts.min.js"></script>)

  1. 创建容器
<div id="chart"></div>
  1. 指定容器大小
#chart {
    width: 800px;
    height: 400px;
}
  1. 初始化echart实例
const myChart = echarts.init(document.getElementById('chart'))
  1. 指定配置选项,详细选项可以参考文档。
myChart.setOption({
    // 标题
    title: {
        text: '我的第一个图表',
    },
    // 图例
    legend: {
        type: 'plain',
        show: true,
        
    },
    // 提示
    tooltip: {},
    // x轴数据
    xAxis: {
        // 添加data选项，会默认认为type为 category 离散数据类目图
        data: ['x1', 'x2', 'x3']
    },
    yAxis: {
    },
    series: [
        {
            name: 'y1',
            // 类型
            type: 'bar',
            data: [1, 2, 3],
            
        },
        {
            name: 'y2',
            type: 'bar',
            data: [3, 4, 5]
        }
    ]
    
})

三、vue集成

  • 自定义一个echarts组件
<template>
  <div :id="chartId" :style="{ width: '100%', height: '300px' }"></div>
</template>

<script>
  import * as echarts from 'echarts';
  
  export default {
    name: 'Echarts',
    props: ['chartId', 'option'],
    mounted() {
      //挂载的时候实例化echart对象，并配置父元素传来的选项
      this.drawLine();
    },
    methods: {
      drawLine() {
        // 基于准备好的dom，初始化echarts实例
        let myChart = echarts.init(document.getElementById(this.chartId));
        // 绘制图表
        myChart.setOption(this.option);
      },
    },
  };
</script>
