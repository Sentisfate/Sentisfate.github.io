Excalidraw 本地部署

2023-07-14 01:46

https://excalidraw.com/

一个非常好用的开源在线画板工具，由于公司防火墙的原因被墙，因此决定本地部署一个。步骤如下：

  1. 下载源码，https://gitee.com/mirrors/Excalidraw
  2. 安装依赖，首先确保 yarn和npm（14以上）已安装，进入项目根目录安装依赖npm i
  3. npm run start查看是否可以运行
  4. npm run build打包项目
  5. 使用nginx部署打包后的项目

  1. 配置nginx配置文件，选择你喜欢的端口

  server {
     listen       16667;
     server_name  localhost;

     location /excalidraw {
         root   html;
         index  index.html index.htm;
     }
  }
  1. 重启nginx nginx -s reload后访问你配置的地址

这样就可以愉快的离线使用了，再也不用担心被墙了。当然你可以选择将其部署到你的任意一个服务器上，操作同理。
