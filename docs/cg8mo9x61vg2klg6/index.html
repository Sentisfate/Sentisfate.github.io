Teleport

2023-02-22 05:38

teleport 是vue的一个内置组件。其功能是将teleport内的模板传送到DOM结构的外层。下面是一个示例：

<template>
  <div class="outer">
    <h3>Tooltips with Vue 3 Teleport</h3>
    <div>
      <button @click="open = true">Open Modal</button>

      <Teleport to="body">
      <div v-if="open" class="modal">
        <p>Hello from the modal!</p>
        <button @click="open = false">Close</button>
      </div>
      </Teleport>

    </div>
  </div>
</template>
<script setup>
import { ref } from 'vue';

const open = ref(false)
</script>
<style>
.modal {
  position: fixed;
  z-index: 999;
  top: 20%;
  left: 50%;
  width: 300px;
  margin-left: -150px;
}

.outer {
  background: pink;
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
</style>

我们希望对话框和按钮同属于一个组件(因为在逻辑上具有强相关性)。但是，此时如果修改最外层的盒子的样式时，模态框的样式可能也会发生变化。当加上teleport时，指定一个to属性来将内部的模板传递到指定标签下。通常是body标签(不会发生变化)。可以设置下面两个属性。to可以指顶htmlelement 或者选择器。必填disabled: boolean 是否禁用传送。true的话，将保留在原来的位置，不会传送。可动态修改
