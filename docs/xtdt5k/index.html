其实base64和base64url的区别也就在这里。 base64编码结果中会有+、/、=三个特殊字符，它们在url中属于特殊字符是直接无法传递的； base64url其实就是把字符中的’+‘和’/‘分别替换成’-‘和’_’，另外把末尾填充的‘=’去掉;其他都一样。
