接雨水

2022-09-09 09:50

题目描述

解法1 动态规划

思路：

木桶效应，每个格子能接到的水由它两边最高的一块板中较短的一个决定。 可以用一个数组dp[i]来记录每个格子能接到的最高的雨水的数量，最后求和即可

临界条件： :::info 第一个和最后一个接不到水 :::

/**
 * @param {number[]} height
 * @return {number}
 */
var trap = function(height) {
    // 对于每个格子来说，它所能接到的水类似于木桶效应，取决于左右两边最高的一块板的较小的一个减去它本身的高度
    //如果左边或者右边没有比它高的，就接不到水
    //动态规划 dp i 记录每个格子能接到的水
    //第一个和最后一个接不到水
    const dp=[0]
    let len=height.length
    for(let i=1;i<len-1;i++){
        //这里截取的包括了自身，是为了方便计算
        let leftMax=Math.max(...height.slice(0,i+1))
        let rightMax=Math.max(...height.slice(i))
        dp[i]=Math.min(leftMax,rightMax)-height[i]
    }
    let sum=0
    dp.forEach(item=>{sum+=item})
    return sum
};

解法2 记忆化+动态规划

解法1没有考虑到最高和最低的复用性，因此可以加入记忆来提高效率思路：

在解法1的基础上，每次比较时，找到当前左右两边的最高点，再移动到下一个格子时，只需比较上一次的最大值和当前格子的大小即可，并累加当前格子的接水量

  • 从左往右遍历的话
  • 对于左边而言，每次比较的是上一次的leftMax和当前格子的高度，如果当前的格子高于leftMax，leftMax更新
  • 对于右边而言，每次比较上一次的rightMax和上一个格子，如果上一个格子低于rightMax的话，rightMax不变，否则，寻找新的rightMax
/**
 * @param {number[]} height
 * @return {number}
 */
var trap = function(height) {
    // 对于每个格子来说，它所能接到的水类似于木桶效应，取决于左右两边最高的一块板的较小的一个减去它本身的高度
    //如果左边或者右边没有比它高的，就接不到水
    //动态规划 dp i 记录每个格子能接到的水
    //第一个和最后一个接不到水
    let len=height.length
    let sum=0
    let leftMax=height[0]
    let rightMax=Math.max(...height.slice(1))
    for(let i=1;i<len-1;i++){
        //这里截取的包括了自身，是为了方便计算
        leftMax=Math.max(leftMax,height[i])
        if(rightMax==height[i]){
            sum+=0
            rightMax=Math.max(...height.slice(i+1))
        }else{
            sum+=Math.min(leftMax,rightMax)-height[i]
        }
    }
    return sum
};

/**
 * @param {number[]} height
 * @return {number}
 */
var trap = function(height) {
    // 找到最大的索引，在其左边的只需比较左边的最大值，在其右边的只需比较右边的最大值
    let len=height.length
    let split=0
    let leftMax=height[0]
    let rightMax=height[len-1]
    let sum=0
    for(let i=0;i<len;i++){
        if(height[split]<height[i]){
            split=i
        }
    }
    //计算左侧的接水量
    for(let i=1;i<split;i++){
        if(height[i]>leftMax){
            leftMax=height[i]
        }
        sum+=leftMax-height[i]
    }
    for(let i=len-2;i>split;i--){
        if(height[i]>rightMax){
            rightMax=height[i]
        }
        sum+=rightMax-height[i]
    }
    return sum
};

解法2 单调栈

思路：

对于接水量，可以看成是一个个宽为w，高位h的矩形。

left和right为左右较高的元素的索引

w,和h可以用一个单调栈**stack**来计算。

  • 栈中存储元素的索引
  • 栈底到栈顶满足**height[stack[i-1]]<height[stack[i]]**
  • 当插入元素的高度大于栈顶元素的高度时，此时就形成了一个接水区间。

  • 将栈顶元素出栈，并计算接水量，接着继续比较新的栈顶元素，与插入元素的关系，直至栈空或者栈顶元素的高度大于插入元素
/**
 * @param {number[]} height
 * @return {number}
 */
var trap = function(height) {
    let len=height.length
    let stack=[0]
    let sum=0
    for(let i=1;i<len;i++){
        let top=stack.length-1
        while(top>=0&&height[stack[top]]<height[i]){
            if(top==0){
                stack.pop()
                break
            }
            let w=i-stack[top-1]-1
            let h=Math.min(height[stack[top-1]],height[i])-height[stack[top]]
            sum+=w*h
            stack.pop()
            top=stack.length-1
        }
        stack.push(i)
    }
    return sum
};
