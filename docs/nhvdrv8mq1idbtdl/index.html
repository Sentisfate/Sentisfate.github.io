🙃 关于webgl语法的更新

2023-03-20 14:16

Cesium 最近一次更新默认使用了webgl2的语法，原来的webgl1的语法不在默认支持了，需要单独配置，否则可能会出现如下的错误： RuntimeError: Fragment shader failed to compile. Compile log: ERROR: 0:1: ‘varying’ : Illegal use of reserved word

见官网：https://github.com/CesiumGS/cesium/blob/1.102/CHANGES.md

最简单的方法是降级，回到之前的版本，亲测1.98无冲突

 "cesium": "^1.98",

想要更长远一点的解决方法，是更新着色器相关代码。

  const vs =
    `in vec3 position3DHigh;
    in vec3 position3DLow;
    // in vec4 color;
    out vec4 v_color;
    in float batchId;
    void main()
    {
        vec4 p = czm_computePosition();
        v_color = vec4(1.0,1.0,0.5,1.0);
        p = czm_modelViewProjectionRelativeToEye * p;
        gl_Position = p;
        gl_PointSize = 10.0;
    }
    `;
  //定义片元着色器
  const fs =
    `
    in vec4 v_color;
    void main()
    {
         float d = distance(gl_PointCoord, vec2(0.5,0.5));
         if(d < 0.5){
            out_FragColor = v_color;
         }else{
            discard;
         }
    }
    `;
