集成cesium

2022-12-10 12:38

现在应该可以直接设置node_modules里的路径

步骤

  1. 创建react项目并安装相应依赖，清空默认内容
  2. npm i cesium下载cesium的包
  3. 导入cesium的样式和依赖
import 'cesium/Build/Cesium/Widgets/widgets.css';
import * as Cesium from 'cesium';
  1. 获取cesium的token
Cesium.Ion.defaultAccessToken = token;
  1. 在react创建地图容器，并设置样式
  2. 设置CESIUM_BASE_URL，路径'/'默认为public文件夹下，需要将node_modules/cesium/Build/Cesium文件夹复制到public文件加下，设置CESIUM_BASE_URL='/Cesium'
window.CESIUM_BASE_URL = '/Cesium';
  1. 创建viewer对象
const viewer = new Cesium.Viewer('map');

完整代码

import { useState, useEffect } from 'react';
import './App.css';
import { token } from './token/cesium-token';
import 'cesium/Build/Cesium/Widgets/widgets.css';
import * as Cesium from 'cesium';
Cesium.Ion.defaultAccessToken = token;
function App() {
  useEffect(() => {
    window.CESIUM_BASE_URL = '/Cesium';

    const viewer = new Cesium.Viewer('map');
    viewer.infoBox.frame.removeAttribute('sandbox');
    viewer.infoBox.frame.src = 'about:blank';
  }, []);
  return <div id="map"></div>;
}

export default App;
