关于像素不得不知道的那点事

2022-11-21 10:49

概念

  • 英寸，寸，长度单位，固定不变，绝对的
  • 像素 图片的最小单元格
  • 分辨率 一个图像的像素的个数可以理解成分辨率，例如30.72万像素可以表达成640x 480，屏幕分辨率则说明屏幕最多能划分为多少个像素
  • 物理像素与逻辑像素
    • 物理(设备)像素**dp(device pixels)**指设备实际拥有的像素点，即设备的分辨率
    • 逻辑(设备独立)像素**dip(device independent pixels)**，指图片的尺寸大小，其本身的分辨率不变，但是外观尺寸发生改变，如放大缩小等。css中的px也是逻辑像素

设备像素比 dpr

dp/dip称为设备像素比，简称dpr一般电脑，设备像素与逻辑像素相等dpr=1，高分辨率的电脑，dpr会大一些，即一个逻辑像素对应多个设备像素，可以通过浏览器的**window.devicePixelRatio**来查看。

那么，为什么不把dpr设为1呢

假设dpr为1，则高分辨率的屏幕相对于低分辨率的屏幕，显示同一张图片，外观尺寸看起来会更小。这显然不是特别合理。因此为了在不同分辨率设备上显示图片尺寸一致，(也就是图像的逻辑像素的宽高要一定)。例如一张图像的分辨率为w x h，在同样分辨率的设备上，dpr设为1。即1个dip对应一个dp，而在2w x 2h的设备上显示时，dpr应为**0.5**，即一个dip对应4(2*2)个dp的大小。此外dpr并非一成不变，是可以改变的，比如浏览器放大或者使用放大镜时，dpr就被缩小了。
