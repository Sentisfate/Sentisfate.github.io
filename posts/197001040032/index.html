PicGO 下载地址：GitHub - Molunerfinn/PicGo: A simple & beautiful tool for pictures uploading built by vue-cli-electron-builder

  1. github 创建一个仓库，用于存储图床
  2. github 新建一个 token，记得保存，后面会用

  1. 进入 picGO 的图床配置界面进行配置
    1. 仓库名格式 [github用户名]/[仓库名]
    2. 分支，github 创建仓库里的分支，默认主分支为 main
    3. 设定 token，前面创建的 token
    4. 指定存储路径
    5. 自定义域名格式https://cdn.jsdelivr.net/gh/[github用户名]/[仓库名]@main 使用了 jsdelivr 作为 cdn 加速

以上步骤完成后去上传区测试即可了解是否上传成功了，成功后相册处可以进行复制 url
